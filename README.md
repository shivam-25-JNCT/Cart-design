# 🛒 React Cart System (Context API + Tailwind CSS)

A beginner-friendly e-commerce cart system built using **React**, **Context API**, and **Tailwind CSS**.

This project demonstrates how to manage global state using Context API and perform cart operations like adding, removing, incrementing, decrementing items, and calculating total quantity.

---

## 🚀 Features

* ✅ Add to Cart
* ✅ Remove from Cart
* ✅ Increase / Decrease Quantity
* ✅ Auto remove item when quantity becomes 0
* ✅ Total Items Counter (Navbar Badge)
* ✅ Total Price Calculation
* ✅ Global State Management using Context API
* ✅ Clean UI with Tailwind CSS

---

## 🧠 Concepts Used

* React Hooks (`useState`, `useContext`)
* Context API
* Derived State (`reduce` for total items & total price)
* Conditional Rendering
* Array Methods (`map`, `filter`, `reduce`, `findIndex`)
* Functional State Updates
